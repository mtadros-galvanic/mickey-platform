# Ansible

This directory owns the post-install host baseline and guest bootstrap for `mickey-platform`.

## Playbooks

- `playbooks/proxmox-host.yml`
  - baseline packages required for storage setup
  - create and mount the `bulk` storage on the 4 TB disk
  - register the `bulk` directory storage with Proxmox
- `playbooks/control-vm.yml`
  - bootstrap the tiny automation VM
  - install Terraform, SOPS, Ansible, and supporting tools
- `playbooks/desktop-vm.yml`
  - bootstrap the main Ubuntu Desktop VM
  - install XRDP, Docker, Samba, and guest-agent packages
  - format and mount the extra data disk at `/srv/share`
  - export the `mickey-share` Samba share
- `playbooks/site.yml`
  - imports the three playbooks

## Inventory

- `inventory/hosts.yml`
  - tracked static inventory containing the Proxmox host
- `inventory/hosts.generated.yml`
  - generated by Terraform and ignored by Git

Run guest playbooks with both inventories.
